# Stage 6 Prompts: Capstone

Topics: capstone selection, report, interview prep

## Planning prompt

```text
Read AGENTS.md, COURSE_SYLLABUS.md, RUBRIC.md, and the relevant lesson README.
Create a staged plan for this stage.
Do not implement final optimized code yet.
Tell me the smallest next task and the definition of done.
```

## Review prompt

```text
Review my work for this stage against RUBRIC.md.
Score each category 0-5, identify the weakest gap, and recommend the next concrete task.
```

## Quiz prompt

```text
Quiz me on this stage, one question at a time.
Focus on concepts I need to explain in an interview.
After each answer, grade it and tell me what to review.
```
